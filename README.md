## Primeros pasos

- Instalamos AngularJs
  - Directo de la página, (puedes descargarlo)
- Instalamos Node.js
  - Desde la página oficial
  - En consola: **_npm init -y_**
  - **_npm i -g express-generator_**
  - Creamos un servidor con EXPRESS:
    -  **_express myserver_**
    -  **_npm install_**
- Instalamos Grunt
